# admin-dashboard

## Description

A full stack application dashboard where users can sign up using their email addresses. Admins can also view users and assign roles and access for each user  

## Tech Stack
1. React.js
2. Nest.js
3. Firebase 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
